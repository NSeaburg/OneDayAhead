Embedding the Learning Experience

This page demonstrates how to embed our interactive AI-powered learning platform into your own website. The example below shows the learning experience embedded in an iframe.

How to Embed

To embed this learning experience in your own website, simply add the following HTML code to your page. Make sure to replace YOUR_DEPLOYMENT_URL with the actual URL where this application is deployed.

<iframe src="YOUR_DEPLOYMENT_URL/embed.html" style="width: 100%; height: 600px; border: none;" title="Interactive Learning Experience" allow="microphone; camera" ></iframe>

Customization Options

You can customize the appearance of the embedded learning experience by modifying the following parameters:

  • Height: Change the height value in the style attribute to fit your layout
  • Width: Adjust the width percentage to control how much space the iframe takes up
  • Border: Add a border by changing the border style property

Advanced Integration

For more advanced integration scenarios, you can also communicate with the learning application using cross-window messaging:

// Listen for messages from the learning application window.addEventListener('message', function(event) { // Make sure the message is coming from your application domain if (event.origin === 'YOUR_DEPLOYMENT_URL') { console.log('Message received from learning app:', event.data); // Handle different message types if (event.data.type === 'completion') { // Handle completion event console.log('User completed the learning module!'); } } });
