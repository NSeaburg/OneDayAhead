How to Embed the Learning Platform

This guide explains how to embed our learning platform in your website or learning management system (LMS).

Note: Due to security settings, embedding in certain environments may require additional configuration. Please contact support if you encounter any issues.

Method 1: Direct Link

The simplest approach is to provide a direct link to the application:

<a href="https://new-venture-nate127.replit.app" target="_blank">Open Learning Platform</a>

This will open the application in a new browser tab.

Method 2: HTML Download

Download our embedding template and host it on your own server.

This template includes everything needed to properly embed the learning platform in an iframe.

Method 3: Standalone Page

Use our standalone embedding page:

<iframe src="https://new-venture-nate127.replit.app/iframe-app" 
       style="width: 100%; height: 600px; border: none;" 
       title="Interactive Learning Experience"></iframe>

This page is specifically designed for embedding with appropriate security headers.

Troubleshooting

If you're experiencing issues with embedding:

  • Ensure your site uses HTTPS if you're embedding from an HTTPS source
  • Check your Content Security Policy settings if you have them configured
  • Try Method 1 (direct link) if iframe embedding doesn't work in your environment
  • Contact support for additional assistance
